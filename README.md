# 🎮 Snake Game (JavaScript)

A classic Snake Game built using HTML, CSS, and JavaScript. This project delivers a smooth and interactive gaming experience directly in the browser.

## 🚀 Live Demo
https://abhishekyadav071.github.io/snake-game-js/

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features
- 🎯 Classic snake gameplay
- ⚡ Smooth controls and animations
- 📱 Responsive design
- 🔄 Real-time score tracking
- 💥 Game over detection

## 🎮 How to Play
- Use Arrow Keys to control the snake
- Eat food to grow
- Avoid walls and self-collision

## 📁 Project Structure
- index.html
- style.css
- script.js
